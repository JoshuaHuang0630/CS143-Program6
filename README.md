# Josephus Problem
The Josephus Problem is a math problem where people standing in a circle are eliminated according to a given count until there is only one person remaining.

This program will offer a simulation for the progression of the elimination. It will also provide a mathematical proof to verify the result.

The simulation will begin with a circle of people, whose names will be read from a file. User input decides the elimination count.

The simulation also comes with a UI option, which visualizes the progression of the simulation.

Note: Code in this solution will be completed by AI
